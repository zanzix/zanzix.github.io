First we will define an intrinsically typed language

evalTy : Ty -> Type

evalTy N = Nat

evalTy U = ()

evalTy (Imp t1 t2) = (evalTy t1) -> (evalTy t2)

evalTy (Prod t1 t2) = (evalTy t1, evalTy t2)

evalTy (Sum t1 t2) = Either (evalTy t1) (evalTy t2)
